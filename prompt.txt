MINI PROJECT: 2D GRAPHICS EDITOR IN C

====================================================
PROMPT 1
========

Write a menu-driven 2D graphics editor in C using a 2D character array. The editor should display graphics using '*' and '_' characters.

# RESPONSE

Generated the basic project structure with:

* Canvas represented as a 2D array
* Menu-driven interface
* Functions to initialize and display the canvas

====================================================
PROMPT 2
========

Create a rectangle drawing function in C that draws the border of a rectangle on a 2D character canvas using '*' characters.

# RESPONSE

Generated drawRectangle() function that:

* Accepts x, y, width, and height
* Draws rectangle borders
* Performs boundary checking

====================================================
PROMPT 3
========

Create a line drawing function in C for a text-based graphics editor. The function should draw a line between two points on a character canvas.

# RESPONSE

Generated drawLine() function using DDA (Digital Differential Analyzer) algorithm:

* Takes start and end coordinates
* Calculates increments
* Plots points using '*'

# RESPONSE

* Uses midpoint circle algorithm
